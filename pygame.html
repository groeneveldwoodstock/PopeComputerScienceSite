Objective: Practice reading and navigating a multi-file application to understand, edit, and improve on it.

Step 1: Run the Pygame program below.

☑️ Look at the file structure and how different files are imported in the beginning of other files.

☑️ Read the ReadMe.txt and see the suggested improvements.

☑️ When you are making changes make sure to comment the code and add all changes to the ReadMe.txt file.

You can open the starter code below as well. Fork it to add it to your Sandbox in CodeHS.
