Welcome to Pope Computer Science

This site was designed to explain the computer science course offerings at Pope High School. Select a course above for a description of the course and the current calendars. All work done in class is posted on the calendar for that course. Links to calendars are posted below the course name (click the name for a description of the class). Most calendar links require access to the CodeHS platform and enrollment in the course. Parents can use the calendar to see what was assigned on a given day. Students can access their assignments through the calendar.
